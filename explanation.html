Explanation of Selectors and CSS Inclusion

Selectors Used:

  • Element selector: Used for h1, p, and button elements to apply general styles.
  • Class selector: Used for .card to style the main container div.
  • ID selector: Used for #student-name to style the h1 element specifically.
  • Descendant selector: Used for .card p to style paragraphs inside the card.

CSS Inclusion:

  • Internal CSS: Used in the <style> tag within the HTML head for main styling of .card, h1, and .card p.
  • Inline CSS: Used on the h1 element with the style attribute for text-align: center.
  • External CSS: Used in the linked student_card.css file for styling the button.
